Чтобы установить инструмент venv-создает виртуальное окружение(изолированное пространство)
	Для убунту 
		sudo apt install python3-venv  

	Для мак
		brew install viruelenv 
		
Что бы создать в проекте вируальное окружение
	python3 -m venv название_виртуального_окружения(чаще всего называют venv либо env)
	

Чтобы активировать виртуальное окружение
        . venv/bin/activate
        

Чтобы устанавливать какието библеотеки, модули, пакеты
	pip3 install название_библеотеки(на прошлом уроке установили bs4 и requests)
