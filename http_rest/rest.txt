API - (application programming interface) - то как должен выглядить проект

REST - (represetational state transfer) - архитектурный стиль, где идет работа между клиентом и сервером. API следует REST

Принципы REST:
1. Единый интерфейс
2. Разграничение клиента и сервера
3. Многоуровневая система
4. Нет сохранение состояние
5. Кэширование
6. Код предоставляется по запросу

RESTfullAPI - Api, которая полностью соответствует правилам REST
